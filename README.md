Boggle Solver
=============

This is a standalone webapp for solving boggle.

Running
-------
Project is written in kotlin and build with gradle. To run:
```
$ ./gradlew run
```

Then visit: http://localhost:8080

Fill in a 16 character board and click "Solve" to see the results.
